# Over the Celsus

PC | 턴제 퍼즐 게임 | Cocos2d-x | c++ | (2016)

4개 스테이지로 구성된 퍼즐 게임.

이어진 보석들을 클릭하여 제한된 턴 내에 엘릭서 게이지를 가득 채우고 많은 점수를 얻는 것이 목표이다.

스테이지 별 점수는 합산되며, 제한된 턴 내에 엘릭서 게이지를 가득 채우지 못하면 게임오버.
